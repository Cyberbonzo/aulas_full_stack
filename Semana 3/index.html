Contar até um número · laço while

🗣 Linguagem Natural

  1. Pedir ao usuário que informe um número inteiro positivo.
  2. Verificar se o valor é realmente positivo. Se não for, avisar e encerrar.
  3. Iniciar um contador em 1.
  4. Enquanto o contador for menor ou igual ao número informado, exibir o valor do contador e incrementá-lo em 1.
  5. Ao final do laço, a contagem estará completa.

📋 Pseudocódigo

INÍCIO
  LEIA numero
  SE numero <= 0 ENTÃO
    ESCREVA "Por favor, informe um número positivo."
    FIM
  FIM_SE
  contador ← 1
  ENQUANTO contador <= numero FAÇA
    ESCREVA contador
    contador ← contador + 1
  FIM_ENQUANTO
FIM

💻 Código JavaScript

// Exercício 1 — Contar até um número (laço while)
// ════════════════════════════════════════════════════════════════

// FUNCTION: função em JavaScript é um bloco de código reutilizável.
// Quando você define uma função, o código não executa imediatamente.
// Executa apenas quando a função é CHAMADA (invocada).

function contarAteNumero() {
  // prompt() exibe uma caixa de diálogo pedindo ao usuário um valor.
  // parseInt() converte o texto digitado em um NÚMERO INTEIRO.
  // Exemplo: prompt retorna "5" (texto), parseInt converte para 5 (número).
  const numero = parseInt(prompt("Digite um número inteiro positivo:"));

  // VALIDAÇÃO — Verifica se o valor é válido.
  // isNaN() retorna TRUE se o valor NÃO é um número.
  // NaN = "Not a Number" (não é um número).
  //
  // Exemplo: isNaN("abc") = true (não é número)
  //          isNaN("123") = false (é número)
  if (isNaN(numero) || numero <= 0) {
    // Se a validação falhar, retorna mensagem de erro.
    // return encerra a execução da função e retorna um valor.
    return "Valor inválido. Por favor, informe um número inteiro positivo.";
  }

  // VARIÁVEL RESULTADO — Acumula toda a contagem como texto.
  // Template literal (crase ``) permite inserir variáveis com ${}.
  // \n = caractere de quebra de linha (pula uma linha).
  let resultado = `Contagem de 1 até ${numero}:\n`;

  // CONTADOR — Variável que controla o loop.
  // Iniciada em 1 (por isso let, não const, pois vai mudar).
  let contador = 1;

  // LAÇO WHILE — "Enquanto" (enquanto condição é verdadeira).
  // while (condição) { bloco de código }
  //
  // Funcionamento:
  // 1. Verifica se a condição é TRUE.
  // 2. Se sim, executa o bloco { }.
  // 3. Volta ao passo 1, verifica novamente.
  // 4. Se condição fica FALSE, sai do loop.
  //
  // IMPORTANTE: Você DEVE garantir que a condição fica FALSE eventualmente.
  // Senão, o loop roda infinitamente (travamento).
  while (contador <= numero) {
    // += é um operador de atribuição composta.
    // resultado += contador  é igual a:  resultado = resultado + contador
    // Adiciona o valor do contador ao final da string resultado.
    resultado += contador + "  ";

    // ++ é o operador de incremento.
    // contador++ é igual a: contador = contador + 1
    // Aumenta o contador em 1.
    //
    // CRÍTICO: sem este incremento, o loop rodaria infinitamente
    // porque contador sempre seria <= numero.
    contador++;
  }

  // Retorna a string com toda a contagem acumulada.
  return resultado;
}

Soma de números · laço for

🗣 Linguagem Natural

  1. Solicitar ao usuário um número inteiro positivo.
  2. Criar uma variável acumuladora, iniciada em zero.
  3. Usar um laço for que vai de 1 até o número informado.
  4. A cada repetição, somar o valor atual do laço ao acumulador.
  5. Ao terminar o laço, exibir o resultado final da soma.

📋 Pseudocódigo

INÍCIO
  LEIA numero
  soma ← 0
  PARA i DE 1 ATÉ numero FAÇA
    soma ← soma + i
  FIM_PARA
  ESCREVA "A soma de 1 até", numero, "é:", soma
FIM

💻 Código JavaScript

// Exercício 2 — Soma de números (laço for)
// ════════════════════════════════════════════════════════════════

// FOR LOOP — Um dos loops mais comuns em programação.
// O FOR é ideal quando você SABE quantas vezes quer repetir.
//
// Sintaxe:
// for (inicialização; condição; incremento) {
//   bloco de código
// }
//
// Inicialização: executada UMA VEZ antes do loop começar.
// Condição: verificada ANTES de cada repetição.
// Incremento: executado APÓS cada repetição (antes de verificar novamente).

function somarNumeros() {
  const numero = parseInt(prompt("Digite um número inteiro positivo:"));

  // Validação — mesma lógica do exercício anterior.
  if (isNaN(numero) || numero <= 0) {
    return "Valor inválido. Por favor, informe um número inteiro positivo.";
  }

  // ACUMULADOR — Variável que acumula valores ao longo do loop.
  // Iniciada em 0 (elemento neutro para soma).
  let soma = 0;

  // FOR LOOP em detalhes:
  //
  // let i = 1;        ← Inicialização: i começa em 1
  // i <= numero;      ← Condição: continua enquanto i <= numero
  // i++               ← Incremento: aumenta i em 1 após cada iteração
  //
  // Fluxo de execução para numero = 5:
  // Repetição 1: i=1, 1<=5? SIM, executa bloco, depois i++ (i=2)
  // Repetição 2: i=2, 2<=5? SIM, executa bloco, depois i++ (i=3)
  // Repetição 3: i=3, 3<=5? SIM, executa bloco, depois i++ (i=4)
  // Repetição 4: i=4, 4<=5? SIM, executa bloco, depois i++ (i=5)
  // Repetição 5: i=5, 5<=5? SIM, executa bloco, depois i++ (i=6)
  // Verificação: i=6, 6<=5? NÃO, sai do loop.
  for (let i = 1; i <= numero; i++) {
    // += é operador de atribuição composta.
    // soma += i   é igual a:  soma = soma + i
    //
    // Exemplo com numero = 5:
    // i=1: soma = 0 + 1 = 1
    // i=2: soma = 1 + 2 = 3
    // i=3: soma = 3 + 3 = 6
    // i=4: soma = 6 + 4 = 10
    // i=5: soma = 10 + 5 = 15
    soma += i;
  }

  // Retorna o resultado final da soma.
  return `A soma de 1 até ${numero} é: ${soma}`;
}

Validação de número · função + laço while

🗣 Linguagem Natural

  1. Criar uma função responsável por ler e validar o número.
  2. Dentro da função, pedir ao usuário um número entre 1 e 60.
  3. Verificar se o valor está dentro do intervalo válido.
  4. Enquanto o número for inválido, exibir uma mensagem de erro e pedir novamente.
  5. Quando o número for válido, retorná-lo e exibir uma confirmação.

📋 Pseudocódigo

FUNÇÃO lerNumeroValido()
  LEIA numero
  ENQUANTO numero < 1 OU numero > 60 FAÇA
    ESCREVA "Número inválido! Digite um valor entre 1 e 60."
    LEIA numero
  FIM_ENQUANTO
  RETORNE numero
FIM_FUNÇÃO

INÍCIO
  numeroValido ← lerNumeroValido()
  ESCREVA "Número válido recebido:", numeroValido
FIM

💻 Código JavaScript

// Exercício 3 — Validação de número com função (laço while)
// ════════════════════════════════════════════════════════════════

// ENCAPSULAMENTO — A lógica de validação fica em sua própria função.
// Vantagem: reutilizável, legível, fácil de manter.
//
// Função 1: lerNumeroValido()  — Responsável APENAS por validação
// Função 2: validarNumero()   — Chama a primeira e exibe resultado

// ESCOPO — Variáveis dentro de uma função existem apenas lá.
// "numero" só existe dentro de lerNumeroValido().
// Funções externas não podem acessá-lo direto.

// Função dedicada a ler e validar entrada do usuário.
function lerNumeroValido() {
  // Pede o número ao usuário.
  let numero = parseInt(prompt("Digite um número entre 1 e 60:"));

  // WHILE + LÓGICA DE VALIDAÇÃO
  // Continue pedindo enquanto a entrada for inválida.
  //
  // Condição de invalidez:
  // • isNaN(numero)    — não é um número
  // • numero < 1       — menor que 1
  // • numero > 60      — maior que 60
  //
  // OPERADOR ||  (OR — "ou")
  // Retorna TRUE se QUALQUER uma das condições for TRUE.
  // Exemplo: isNaN(numero) || numero < 1 || numero > 60
  //          Retorna TRUE se QUALQUER Uma for verdadeira.
  //
  // OPERADOR &&  (AND — "e")
  // Retorna TRUE APENAS SE todas as condições forem TRUE.
  // Exemplo: numero >= 1 && numero <= 60
  //          Retorna TRUE APENAS SE ambas forem verdadeiras.
  while (isNaN(numero) || numero < 1 || numero > 60) {
    // Se chegou aqui, o número é inválido.
    // Mostra mensagem de erro e pede novamente.
    numero = parseInt(prompt("Número inválido! Digite um valor entre 1 e 60:"));
  }

  // Quando sai do while, numero GARANTIDAMENTE é válido (entre 1 e 60).
  // IMPORTANTE: return encerra a função IMEDIATAMENTE.
  // Código abaixo do return não executa.
  return numero;
}

// Função que chama lerNumeroValido() e exibe o resultado.
function validarNumero() {
  // Chama lerNumeroValido() e recebe o número válido de volta.
  // numeroValido = resultado da função lerNumeroValido().
  const numeroValido = lerNumeroValido();

  // Retorna mensagem de confirmação.
  return `Número válido recebido: ${numeroValido} ✓`;
}

Desafio — Tabuada · laço for

🗣 Linguagem Natural

  1. Pedir ao usuário um número inteiro.
  2. Usar um laço for que vai de 1 a 10.
  3. A cada repetição, multiplicar o número informado pelo valor atual do laço.
  4. Exibir cada linha da tabuada no formato: "N × i = resultado".
  5. Ao final do laço, a tabuada completa estará exibida.

📋 Pseudocódigo

INÍCIO
  LEIA numero
  PARA i DE 1 ATÉ 10 FAÇA
    resultado ← numero * i
    ESCREVA numero, "×", i, "=", resultado
  FIM_PARA
FIM

💻 Código JavaScript

// Exercício 4 — Tabuada (laço for)
// ════════════════════════════════════════════════════════════════

// Este exercício integra:
// • Entrada de dados (prompt)
// • Validação
// • Loops (for)
// • Formatação de string (template literals)
// • Método string (padStart para alinhamento)

function gerarTabuada() {
  // Pede o número para o qual gerar a tabuada.
  const numero = parseInt(prompt("Digite um número para ver a tabuada:"));

  // Validação simples.
  if (isNaN(numero)) {
    return "Valor inválido. Por favor, informe um número inteiro.";
  }

  // ACUMULADOR DE STRING
  // Ao longo do loop, concatena (junta) todas as linhas da tabuada.
  // \n = quebra de linha.
  let tabuada = `Tabuada do ${numero}:\n`;
  tabuada += "─────────────────\n";
  // Linha decorativa de separação (caractere Unicode).

  // FOR LOOP de 1 a 10.
  // Cada iteração representa uma linha da tabuada.
  //
  // Exemplo: numero = 7
  // i=1: 7 × 1 = 7
  // i=2: 7 × 2 = 14
  // ...
  // i=10: 7 × 10 = 70
  for (let i = 1; i <= 10; i++) {
    // Calcula o resultado (numero multiplicado por i).
    const resultado = numero * i;

    // Template literal com:
    // • ${numero} — o número base
    // • ${i.toString().padStart(2)} — número formatado com padding
    //   .toString() converte número em string
    //   .padStart(2) alinha com espaços à esquerda (mínimo 2 caracteres)
    //   Exemplo: 1 vira "01", 10 vira "10" — alinha perfeitamente
    // • ${resultado} — resultado da multiplicação
    //
    // Exemplo com numero=7:
    // Para i=1: "7 × 01 = 7"
    // Para i=9: "7 × 09 = 63"
    // Para i=10: "7 × 10 = 70"
    tabuada += `${numero} × ${i.toString().padStart(2)} = ${resultado}\n`;
  }

  // Retorna a tabuada completa acumulada.
  return tabuada;
}
