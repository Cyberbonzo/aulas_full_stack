👁️ Mostrar/Esconder Seção


Seção Oculta

Esta seção pode ser escondida clicando no botão acima!

O JavaScript adiciona a classe "escondida" que utiliza CSS (opacity: 0, max-height: 0) para fazer sumir suavemente.

O elemento continua no HTML (DOM), apenas não é visível. Diferente de display: none que remove do fluxo.

📚 Aprendizado Prático

Conceitos DOM aplicados nesta página:

  • getElementById(): Busca elemento único por ID
  • .value: Lê/modifica valor de inputs
  • .textContent: Modifica texto do elemento
  • .classList.add(): Adiciona classe CSS
  • .classList.remove(): Remove classe CSS
  • .classList.toggle(): Alterna classe (add/remove)
  • onclick: Vincula função ao clique do botão
  • Variáveis globais: contador, temaAtivo
  • Transições CSS: Suaviza mudanças visuais
  • Variáveis CSS: --cor-fundo, --cor-texto, etc

💡 Dica: Abra o Console do navegador (F12 → Console) para ver o que acontece internamente quando você interage com a página.
